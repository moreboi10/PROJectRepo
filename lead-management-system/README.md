# Lead Management System (Mini CRM)

## Overview
A full stack web application to manage and track leads.

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL

## Features
- Add lead (name, phone, source)
- View all leads
- Update lead status
- Delete lead
- Search leads
- Dashboard stats

## Setup Instructions

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm start  

## API Endpoints
- POST /api/leads/add
- GET /api/leads
- PUT /api/leads/update/:id
- DELETE /api/leads/delete/:id
